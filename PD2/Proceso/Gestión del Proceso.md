# Gestión de proceso 

Los artefactos clave que dirigieron el proceso fueron la tabla de contribución individual, la bitácora y el uso de reportes. 

Usamos la tabla de contribución para asignar tareas y guiarnos. Además, usando elementos de la metodología Scrum como los sprints, la asignación de roles y reuniones diarias. Estos elementos contribuyeron a la organización del trabajo. 

Para organizar las tareas asignadas y completadas usamos bitácoras que describen todo lo realizado durante las reuniones semanales.

Para el diseño de la página web usamos la inteligencia artificial de DeepSeek resultando en un diseño de media fidelidad.
